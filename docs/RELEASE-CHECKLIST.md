# RELEASE-CHECKLIST.md — 可售发布验收清单（人工回归版）

## 1) 黄金路径（必须全通过）
- 新用户注册 → 登录成功 → Dashboard 可见
- Home Demo：
  - 输入文本 → 点击生成 → 看到流式输出
  - 生成后 Credits 余额减少（按配置）
  - 余额刷新与提示一致
- Credits 不足：
  - 触发拦截（不调用生成或生成被拒）
  - 前端提示清晰
  - 不出现负数余额或异常账本记录

## 2) 权益一致性（必须全通过）
- 任意页面展示的 plan/entitlements 与后端读取入口一致
- Free/Pro 的差异展示不夸大、不写未实现能力
- 所有“是否允许生成”的判断口径一致（同一入口）

## 3) 成本护栏（至少验证两项）
- 匿名额度限制可触发并提示
- 速率限制可触发并提示
- 并发限制可触发并提示
- 异常熔断开关可生效（若实现）

## 4) i18n/SEO（必须全通过）
- en/zh 切换生效，核心文案覆盖
- sitemap/robots/canonical 策略正常
- /readme 默认 noindex 生效

## 5) 对外页面合规（必须全通过）
- Home/Pricing/FAQ/Legal 不出现内部计划、路线图、发布节奏、未上线功能暗示
- 支持邮箱与联系信息一致

## 6) 商品化（建议通过）
- 配置集中：买家改品牌/额度/模型/语言不需要全仓库搜索替换
- 文档可用：陌生人按文档能独立跑通黄金路径
- Demo 易替换：能明确指出“替换 Demo 的位置与步骤”（纯文字说明即可）
